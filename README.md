## Featured Projects

### 🔹 [Automated Talent Analyzer](https://github.com/ananyasd-dev/Automated-Talent-Analyzer)
An AI-powered resume evaluation tool that matches candidate resumes against job descriptions using semantic similarity, and generates structured, LLM-scored skill-gap reports with personalized interview prep tips.
**Tech:** Python, Streamlit, pdfminer, Sentence-Transformers (`all-mpnet-base-v2`), Groq API (`llama-3.3-70b-versatile`)
**Live demo:** [streamlit.app link](https://automated-talent-analyzer-52wtvwmcjvgazfvsaqmpkj.streamlit.app/)

### 🔹 [AlertHer](https://github.com/ananyasd-dev/https://github.com/lathish162/Hackathon1)
A women's safety dashboard combining historical crime data with ML-driven threat analysis — using semantic search over threat reports to surface patterns traditional keyword search misses.
**Tech:** Python, Streamlit, scikit-learn, PyTorch, Sentence-Transformers
**Live demo:** [render.com link](https://hackathon1-687i.onrender.com)
